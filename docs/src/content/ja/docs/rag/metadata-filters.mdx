# メタデータフィルター

メタデータフィルターを使用すると、特定の条件に基づいて検索結果を絞り込むことができます。これにより、より関連性の高い結果を得ることができます。

<Tabs items={['PgVector', 'Pinecone', 'Qdrant', 'Chroma', 'Astra DB', 'LibSQL', 'Upstash', 'Cloudflare', 'MongoDB', 'Couchbase']}>
<Tabs.Tab>
  Couchbaseでは、メタデータフィルターは現在サポートされていません。代わりに、クエリ結果を取得した後にアプリケーション側でフィルタリングを行う必要があります。
